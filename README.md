# Data-Structures-Simulator-C++

A console-based Data Structures Simulator built using C++ that demonstrates the working of fundamental data structures like Stack, Queue, and a basic Sorting Algorithm (Bubble Sort).

This project is designed to strengthen understanding of core Data Structures and Object-Oriented Programming concepts.

🚀 Features
📌 Stack Simulation

Push element

Pop element

Display stack (LIFO order)

📌 Queue Simulation

Enqueue element

Dequeue element

Display queue (FIFO order)

📌 Sorting Demonstration

Accepts user input

Implements Bubble Sort

Displays sorted output

🛠️ Tech Stack

Language: C++

Concepts Used:

Object-Oriented Programming (OOP)

Classes & Objects

STL vector

Menu-driven programs

Basic sorting algorithms

Stack (LIFO)

Queue (FIFO)

🏗️ Project Structure
1️⃣ StackSim Class

Uses vector<int> for storage

Implements:

push()

pop()

display()

2️⃣ QueueSim Class

Uses vector<int> for storage

Implements:

enqueue()

dequeue()

display()

3️⃣ Simulator Class

Demonstrates Bubble Sort

Accepts dynamic user input

Displays sorted result


🧠 Learning Outcomes

This project helps in understanding:

Practical implementation of Stack (LIFO)

Practical implementation of Queue (FIFO)

How Bubble Sort works step-by-step

How vectors can simulate dynamic data structures
